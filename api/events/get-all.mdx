---
title: 'Get All Events'
api: 'GET /events'
---

Retrieve a paginated list of events with optional filtering by category, visibility, type, title, and location type.

## Authentication

No authentication required. Public endpoint.

## Query Parameters

<ParamField query="category" type="string">
  Filter events by category.
  
  **Possible values:**
  - `TECH` - Technology events
  - `SCIENCE` - Science events
  - `MATH` - Mathematics events
  - `BIOLOGY` - Biology events
  - `SPORTS` - Sports events
  - `SOCIAL_SCIENCE` - Social science events
  - `OTHER` - Other categories
</ParamField>

<ParamField query="visibility" type="string">
  Filter events by visibility.
  
  **Possible values:**
  - `PUBLIC` - Publicly visible events
  - `PRIVATE` - Private events
</ParamField>

<ParamField query="type" type="string">
  Filter events by type.
  
  **Possible values:**
  - `SEMINAR` - Seminar events
  - `WORKSHOP` - Workshop events
  - `COMPETITION` - Competition events
  - `BOOTCAMP` - Bootcamp events
</ParamField>

<ParamField query="title" type="string">
  Search events by title (partial match).
</ParamField>

<ParamField query="locationType" type="string">
  Filter events by location type.
  
  **Possible values:**
  - `ONLINE` - Online events
  - `OFFLINE` - Offline/in-person events
</ParamField>

<ParamField query="limit" type="number" default="20">
  Number of events to return per page. Minimum: 1, Maximum: 50.
</ParamField>

<ParamField query="cursor" type="string">
  Cursor for pagination. Use the `nextCursor` value from the previous response to get the next page.
</ParamField>

```bash
curl https://api.dragonhacks.com/events?category=TECH&type=WORKSHOP&limit=10
```

## Pagination

This endpoint uses cursor-based pagination:

1. Make an initial request with optional `limit` parameter
2. Check the `hasNextPage` field in the response
3. If `true`, use the `nextCursor` value in your next request
4. Repeat until `hasNextPage` is `false`

```bash
# First page
curl https://api.dragonhacks.com/events?limit=20

# Next page
curl https://api.dragonhacks.com/events?limit=20&cursor=clx1a2b3c4d5e6f7g8h9i0j1
```
