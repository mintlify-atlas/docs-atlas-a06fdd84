---
title: 'Get Announcements'
description: 'Retrieve all announcements for an event'
---

## Endpoint

<CodeGroup>
```bash cURL
curl https://api.dragonhacks.io/events/{eventId}/announcements
```

```typescript TypeScript
const response = await fetch(
  `https://api.dragonhacks.io/events/${eventId}/announcements`
);
const data = await response.json();
```
</CodeGroup>

## Path Parameters

<ParamField path="eventId" type="string" required>
  The unique identifier of the event
</ParamField>

## Response

<ResponseField name="items" type="array" required>
  List of announcements for the event, ordered by creation date (newest first)
  
  <Expandable title="Announcement Object">
    <ResponseField name="id" type="string" required>
      Unique identifier for the announcement
    </ResponseField>
    
    <ResponseField name="title" type="string" required>
      Title of the announcement
    </ResponseField>
    
    <ResponseField name="content" type="string" required>
      Full content/body of the announcement
    </ResponseField>
    
    <ResponseField name="createdAt" type="date" required>
      Timestamp when the announcement was created
    </ResponseField>
  </Expandable>
</ResponseField>

## Notes

- This endpoint is public and does not require authentication
- Announcements are returned in descending order by creation date
- Only announcements for the specified event are returned
- When announcements are created, approved event registrants receive notifications via email and WhatsApp
